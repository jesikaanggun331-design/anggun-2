# Panduan Formulir Pendaftaran Komunitas Teknologi

Formulir ini digunakan untuk mendaftarkan diri sebagai anggota komunitas teknologi. Isi semua field yang wajib (ditandai dengan *) dengan data yang valid. Berikut panduan singkat untuk mengisi setiap bagian:

## Informasi Pribadi
- **Nama Lengkap**: Masukkan nama lengkap Anda (wajib).
- **Alamat Email**: Masukkan email aktif (wajib, format email valid).
- **Kata Sandi**: Buat kata sandi minimal 8 karakter (wajib).
- **Usia**: Masukkan usia antara 13-100 tahun (wajib).
- **Tanggal Lahir**: Pilih tanggal lahir Anda (wajib).
- **Jenis Kelamin**: Pilih salah satu opsi (wajib).

## Minat
- **Minat**: Pilih minimal satu minat dari opsi yang tersedia (AI, Web, IoT, Game).

## Kontak dan Portofolio
- **Nomor Telepon**: Masukkan nomor telepon dengan format angka 10-13 digit (wajib).
- **Link Portofolio**: Masukkan URL portofolio Anda jika ada (opsional).
- **Upload Foto Profil**: Unggah foto profil dalam format gambar (wajib).

Setelah mengisi, klik **Submit** untuk mengirim data. Jika ingin mengulang, klik **Reset**. Pastikan semua data akurat untuk proses pendaftaran yang lancar.
